## â“ Troubleshooting

### Health Tips Won't Close
**Try:**
1. Tap the X button at top-right
2. If not visible, scroll to the top of the panel
3. Or tap outside the white area
4. Or swipe down on the panel

### Bottom Buttons Not Working
**Solution:**
- Make sure all panels/sheets are closed
- You should see the full dashboard
- No white panels should be covering the screen

### Content Appears Cut Off
**Solution:**
- The panel is scrollable - try scrolling
- If still cut off, close and reopen the panel
- Clear browser cache and reload

### Can't Add to Cart
**Solution:**
- Make sure you're logged in
- Check if medicine is in stock
- Try refreshing the page

### Orders Not Appearing
**Solution:**
- Check notification bell (top-right)
- Go to Medical History â†’ Purchases tab
- Make sure you completed checkout
