## ğŸ› TROUBLESHOOTING

### "Netlify upload fails"
- **Check internet connection**
- Make sure you're dragging the whole folder
- Try using Chrome browser

### "PWA Builder shows errors"
- **Wait 2 minutes after Netlify upload completes**
- Refresh PWA Builder page
- Try your URL directly in browser first

### "APK won't install"
- **Enable "Unknown sources"** (see above)
- Check you have storage space (need ~50 MB)
- Uninstall any old version first

### "App crashes when I open it"
- **Test your Netlify URL in mobile browser first**
- If web version works, APK should too
- Try clearing browser cache and re-generating

### "PWA Builder can't find my app"
- **Check URL is correct** (copy-paste carefully)
- Make sure it starts with `https://`
- Try accessing URL in incognito/private browser
