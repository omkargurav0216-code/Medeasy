## üêõ Troubleshooting

### Common Issues

**"npm: command not found"**
- Install Node.js from https://nodejs.org

**"Build failed"**
- Run `npm install` first
- Check error logs
- See [`DEPLOYMENT_FILES_CHECKLIST.md`](./DEPLOYMENT_FILES_CHECKLIST.md)

**"Site is blank"**
- Check browser console (F12)
- Verify all files uploaded
- Check netlify.toml configuration

**"APK won't install"**
- Enable "Install from unknown sources" on Android
- Check APK isn't corrupted
- Re-download from PWA Builder

**Need more help?**
- Check relevant guide for your issue
- Visit Netlify Community: https://answers.netlify.com
- Check documentation index: [`APK_DOCUMENTATION_INDEX.md`](./APK_DOCUMENTATION_INDEX.md)
